# knativesrv

A minimal experimental HTTP server built with Kotlin/Native, exploring simplicity, explicitness, and low-level control.

## Motivation

Modern web frameworks often introduce multiple abstraction layers and hidden complexity.  
This project explores how a minimal server can be built with a focus on clarity and explicit behavior.

## Features

- Lightweight HTTP server
- Kotlin/Native implementation
- Minimal abstractions
- Focus on simplicity and control

## Design Principles

- **Simplicity over completeness**
- **Explicit over implicit**
- **Low abstraction overhead**

## Why Kotlin/Native?

Kotlin/Native allows building lightweight binaries without the JVM, making it suitable for:
- experimentation with low-level behavior
- performance-oriented exploration
- minimal runtime environments

## Status

This is a **proof of concept** and experimental project.

## Future Ideas
